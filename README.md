#Bingo Generator

##Usage

	bingo.py
		generates a board named bingo.csv

	bingo.py <name>
		generates a board named <name>.csv
		
	bingo.py <name> <boards>
		generates boards named <name>.csv, <name>_2.csv, ..., <name>_<boards>.csv

###Windows
	bingo.py <args>

###MAC-OSX
    python bingo.py <args>
